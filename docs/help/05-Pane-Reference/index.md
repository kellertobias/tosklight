# Pane Reference

Panes are the building blocks of a saved desk. Open a pane from an empty grid cell, move it by its header, and use **Settings** in that header to configure only that pane. The same pane type may appear more than once with different settings.

Every pane has a **Pane Settings** tab. **Grid width** uses 1-24 columns, **Grid height** uses 1-18 rows, and **Remove pane** removes that instance without deleting show data. Pane positions and options are saved in the current user's desk layout.

The following pages document every choice currently offered by **Open Window**, including a screenshot of the pane and its settings:

- [Programming and visualization panes](01-programming-and-visualization.md)
- [Cue and playback panes](02-cues-and-playbacks.md)
- [Utility and diagnostic panes](03-utility-and-diagnostics.md)

Use a full built-in window for temporary focused work. Use a pane when the surface should remain part of a reusable operator layout.
