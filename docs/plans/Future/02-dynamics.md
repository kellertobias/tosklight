# Dynamics

Dynamics is the planned effect engine and is still being conceptualized. It should support applying fixtures to a dynamic, using presets as steps or start and stop values, and configuring how those values are modulated. Where spatial ordering is needed, the 2D stage view may provide the effect grid.

Before implementation, planning needs to settle the data model, editor workflow, playback behavior, and how Dynamics interact with presets, groups, and stored show data.
