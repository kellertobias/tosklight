What it does

  • Command line first. Keypad grammar and muscle memory, matching a real desk.
  • Live programmer. LTP programmer values over HTP playback, with a fixture sheet and 3D stage view.
  • Cue lists and playback. Cue list pool, playback faders, and group masters with page addressing.
  • One desk, many surfaces. Desktop app, browser, keyboard, and attached OSC hardware share one command line.
  • Portable show files. Shows move between desks; desk-level data stays behind.

Builds are not code-signed. On macOS, open the app once via right-click → Open; on Windows, choose “More info” → “Run anyway” at the SmartScreen prompt.

Run the standalone server

Download the server archive for your platform, unpack it, and run light-server. The operator UI is embedded — open http://127.0.0.1:5000. Bind beyond localhost only on a trusted control network, and set LIGHT_DESK_TOKEN when you do.
